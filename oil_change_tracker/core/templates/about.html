About This Project

Welcome to my car oil change tracker project! This is my first project using Django, and I built it to help users keep track of their vehicle maintenance, specifically oil changes. The application allows users to:

  • Register and manage their account.
  • Record and view oil change history for each vehicle.
  • Set reminders for future oil changes based on mileage or date.
  • Manage multiple vehicles and track each one's maintenance separately.

The project is built with Django, a powerful Python framework, and includes features such as user authentication, CRUD operations for vehicle records, and email reminders.
